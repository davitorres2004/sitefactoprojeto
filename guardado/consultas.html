Consulta de Saldo e Extrato de Projetos

Prezado(a) Coordenador(a),

Manter-se informado sobre a saúde financeira do seu projeto é crucial. O Conveniar oferece ferramentas simples para isso.

1. Saldo de Projetos

A consulta de saldo é a maneira mais simples e rápida de identificar o recurso disponível que seu projeto possui.

Detalhes do Saldo

Se o seu projeto tiver várias rubricas, o Conveniar apresentará o saldo individualizado por cada uma, além do saldo total do projeto.

Como Consultar

  1. Acesse o menu "Saldo".
  2. Escolha o seu projeto.
  3. Selecione a data para a qual deseja visualizar o saldo.
  4. Clique em "Consultar" para que o sistema carregue o saldo de cada rubrica e o saldo total.

2. Extrato de Projetos

O extrato do projeto é um relatório detalhado de todas as movimentações financeiras do seu projeto em um período específico. Nele, você pode visualizar analiticamente cada lançamento de entrada ou saída de recurso.

Como Visualizar

  1. Clique no menu "Extrato".
  2. Selecione seu projeto.
  3. Ajuste o intervalo de datas.
  4. Escolha entre Extrato Simplificado ou Extrato Completo:
    • Extrato Simplificado: Não exibe lançamentos estornados nem cancelados.
    • Extrato Completo: Inclui todos os lançamentos, inclusive os estornados e cancelados.

Organização

O extrato é organizado de acordo com as rubricas do projeto. Para análises mais detalhadas, entre em contato com o gestor do seu projeto.

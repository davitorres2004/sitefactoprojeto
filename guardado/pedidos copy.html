1. Processamento dos Pedidos

  • Envio e Salvamento: Os formulários podem ser enviados ao final do processo ou salvos para envio posterior.
  • Imutabilidade após Envio: Uma vez que um pedido é enviado, não pode mais ser alterado. O gestor do projeto na fundação recebe uma pendência para aprová-lo.
  • Informações do Projeto: Em todos os tipos de pedidos, a primeira etapa é selecionar o projeto. O Conveniar então exibe informações cruciais como coordenador, vigência, saldo do projeto e habilita o botão "Plano de Trabalho" que mostra o saldo das rubricas e os itens aprovados. Essas informações são relevantes para sua decisão e para a aprovação do gestor.
  • Número Identificador: Todo pedido salvo pela primeira vez recebe um número identificador. Este número permite rastrear e identificar o pedido no sistema, possibilitando o acompanhamento de todas as etapas de sua tramitação.
Observação: Guarde sempre o número identificador gerado — ele é a referência para acompanhar a tramitação junto à fundação.

Detalhes dos Tipos de Pedidos

Aqui você encontrará informações detalhadas sobre como criar cada tipo de pedido no sistema Conveniar.

1. Pedido de Compra/Serviço

Este pedido é para solicitar à fundação a aquisição de bens materiais ou serviços para o projeto.

  • Como Acessar: Clique no ícone "Pedido de Compra/Serviço".
  • Abas de Preenchimento:
    • Identificação:
      • Projeto: Selecione o projeto para a compra.
      • Rubrica (Opcional): Indique de qual rubrica a compra será debitada.
      • Meta/Etapa (Se cadastrado): Informe a meta/etapa relacionada à compra.
      • Tipo de Compra: Define o tipo de compra a ser realizada.
      • Instituição/Setor: Para qual instituição ou setor a compra será realizada.
      • Finalidade: Qual a finalidade da compra para o projeto.
      • Carência: O sistema informa o tempo médio necessário para o processo de compra (em dias úteis) e verifica se há tempo hábil dentro da vigência do projeto.
    • Local de Entrega: Defina um ou mais locais para a entrega, completando ou alterando o endereço sugerido pelo Conveniar e adicionando telefone e observações.
    • Itens do Pedido:
      • Catálogo de Produtos e Serviços: Use o botão "Procurar" para encontrar itens no catálogo da fundação.
      • Produto não encontrado no catálogo: Marque esta opção e descreva o item detalhadamente.
      • Itens Aprovados: Visualize produtos/serviços previstos no Plano de Trabalho.
      • Descrição: Detalhe ao máximo as características do item para facilitar a compra.
      • Preço Sugerido Unitário: Ajuda o comprador no processo.
    • Sugestão de Fornecedor: Sugira fornecedores por produto, adicionando informações e arquivos anexos que julgar importantes para agilizar a compra.

2. Pedido de Adiantamento

Solicitação de liberação antecipada de recursos para que um favorecido possa pagar despesas de viagens e/ou outras despesas. Verifique restrições no regimento interno da fundação e nas leis do projeto.
